Command What it does Example
pdf1 pdf2 All PDF1 pages, then all PDF2 pages (default) pdf1 pdf2
pdf2 pdf1 All PDF2 pages first, then PDF1 pdf2 pdf1
interleave Alternate pages: P1-pg1, P2-pg1, P1-pg2, P2-pg2 … interleave
reverse Merge both PDFs, then reverse all page order reverse
pdf1:X-Y, pdf2:A-B Custom page ranges from each PDF. Use 'end' for last page. pdf1:1-3, pdf2:2-end
